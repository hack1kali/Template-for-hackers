this port is used for enumeration and usernames validation 
## 1- basic enumeration
```powershell
sudo nmap -n -v -p 25 --script="smtp-* and safe" -oA smtp <% tp.frontmatter["RHOSTS"] %>
```

## 2- enumerate users
```powershell
sudo nmap -n -v -p 25 --script="smtp-enum-users" -oA smtp-users <% tp.frontmatter["RHOSTS"] %>
```

## 3- enumerate each user
If you have collected a a bunch of usernames already, add it to smtp-usernames.txt list and then execute the following command 
```powershell
nano smtp-user.txt  # write what smtp-enum-users told
## use MODE = {EXPN, VRFY, RCPT}
# you can add the hostname if you have it. 
smtp-user-enum -M VRFY -U smtp-user.txt  -t <% tp.frontmatter["RHOSTS"] %> # -D <% tp.frontmatter["VHOST"] %>
smtp-user-enum -M MODE -U smtp-user.txt  -t <% tp.frontmatter["RHOSTS"] %> #-D <% tp.frontmatter["VHOST"] %>
```
