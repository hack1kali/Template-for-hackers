POP is a mail protocol used to get email messages from a remote computer (POP server).
Connect to the server, check information like version...
## 1- POP basic scan
```powershell
nmap -n -v -p 110 -sV --script="pop3-* and safe" -oA pop3 <% tp.frontmatter["RHOSTS"] %>
```


## 2- POP password brute-force
```powershell
hydra -V -f -l USER -P /usr/share/seclists/Passwords/2020-200_most_used_passwords.txt <% tp.frontmatter["RHOSTS"] %> pop3
```

## 3- POP user enum (authenticated)
```powershell
telnet <% tp.frontmatter["RHOSTS"] %> 110 # alternate method  
USER USER@<% tp.frontmatter["VHOST"] %>  
PASS PASS  
LIST # gets list of emails and sizes  
RETR 1 # retrieve first email  
# try real (root) and fake users to see if there is a difference in error msgs
```

## Bruteforce
```powershell
hydra -l <USER> -P /usr/share/seclists/Passwords/2020-200_most_used_passwords.txt -f <% tp.frontmatter["VHOST"] %> pop3 -V
hydra -S -v -l <USER> -P /usr/share/seclists/Passwords/2020-200_most_used_passwords.txt -s 995 -f <% tp.frontmatter["VHOST"] %> pop3 -V
```

### METHODOLOGY

- Connect with netcat or telnet to the POP server. What software and version is it? Searchsploit and google it.
- This service has default credentials or common credentials like admin : admin?
- If we find some users (with finger, smtp, or whatever service), we can try to brute force it. Be carefull, machine could block you.
