
## NFS scan
```powershell
sudo nmap -n -v -p 2049 -sV --script="safe and nfs-*" -oA nfs-scripts <% tp.frontmatter["RHOSTS"] %>
```


## NFS manual enum
```powershell
# list all mountpoints  
showmount -a <% tp.frontmatter["RHOSTS"] %>  
# list all directories  
showmount -d <% tp.frontmatter["RHOSTS"] %> 
# list all exports (remote folders you can mount)  
showmount -e <% tp.frontmatter["RHOSTS"] %>
```

## NFS find no_root_squash (leading to PrivEsc)
```powershell
grep 'no_root_squash' /etc/exports  
grep 'no_all_squash' /etc/exports
```
