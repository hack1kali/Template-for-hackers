### METHODOLOGY

- Has this SSH server any public exploits?
    - CVE-2008-0166 (September 2006 and May 13th, 2008 may be affected). Predictable PRNG Brute Force SSH
    - OpenSSH 2.3 < 7.7 - Username Enumeration - [https://www.exploit-db.com/exploits/45233](https://www.exploit-db.com/exploits/45233)
- Many times the SSH server reveals the Operating System version of the machine. We can look for the OS that support the specific SSH server version. For example, OpenSSH 8.2.p1 Ubuntu 4Ubuntu0.1, corresponds to Ubuntu focal (20.04), according with launchpad [https://launchpad.net/ubuntu/+source/openssh/1:8.2p1-4ubuntu0.1](https://launchpad.net/ubuntu/+source/openssh/1:8.2p1-4ubuntu0.1)
- If we have a potential user, we may brute force it.
