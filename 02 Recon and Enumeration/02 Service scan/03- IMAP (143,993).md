## Enumeration
### Enumerating Mailboxes

Once connected to the IMAP server, you can list available mailboxes using the `LIST` command:
```powershell
a2 LIST "" *
```

```powershell
a3 select INBOX
```
### Enumerating Email Headers
```powershell
a3 FETCH 1 BODY[HEADER.FIELDS (FROM TO SUBJECT DATE)]
```
